# iOSPath

iOSPath, yeni başlayan iOS geliştiricileri için hazırlanmış web tabanlı öğrenme asistanı prototipidir. Proje; öğrenme rotası, kilitli konu ilerlemesi, final portfolyo hazırlığı, kaynak kütüphanesi ve mülakat hazırlık modülü içerir.

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript
- LocalStorage veri yönetimi

## Katmanlı Yapı

- `iospath.html`: Ana panel ve kaynak kütüphanesi
- `roadmap.html`: iOS öğrenme rotası ve teknik yetkinlik ağacı
- `interview.html`: Mülakat hazırlık modülü ve gelişim günlüğü
- `css/iospath.css`: Görsel tasarım ve responsive yapı
- `js/iospath-data.js`: Öğrenme rotası, kaynaklar ve mülakat soruları
- `js/iospath-storage.js`: LocalStorage veri katmanı
- `js/iospath-app.js`: İş mantığı ve dinamik içerik yönetimi

## Çalıştırma

VS Code içinde proje klasörünü açıp `iospath.html` dosyasını tarayıcıda çalıştırabilirsiniz. Live Server eklentisi varsa `Open with Live Server` seçeneğiyle de açılabilir.
